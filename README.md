# Pokemon Guessr

A little game where you try to guess names of random pokemon with pixelated images

### Credits

- Python modules
    - **[Flask](https://palletsprojects.com/p/flask/)**: The webframework hosting the server.

    - **[Flask-SocketIO](https://flask-socketio.readthedocs.io/en/latest/)**: Handle socket.IO requests.

    - **[pypokedex](https://github.com/arnavb/pypokedex)**: Used to get the sprites and names of pokemons.
